Explicação sobre a Mineração de Criptomoedas

O processo de mineraçãoi começa com as transações sendo enviadas à rede e agrupadas em um bloco pelos mineradores. Cada bloco contém uma referência criptográfica ao bloco anterior, formando uma corrente continua e segura, daí o nome "blockchain"

Para adicionar um bloco à cadeia, os mineradores devem encontrar o nome correto que resolve o problema matemático associado ao bloco.

O primeiro minerador que resolver o problema e validar o bloco é recompensado com uma quantidade especifica do ativo virtual, como um metodo de incentivar a mineração e a manutenção da rede.

Uma vez que um bloco é minerado, as transações contidas nele são consideradas validadas. Este bloco é então adicionado ao blockchain público, onde todas as transações são acessíveis e verificáveis por qualquer pessoa na rede.

O que é Blockchain

A BlockChain é um código de acesso publico que toda a BitCoin se baseia, todas transições confirmadas, aumentando a segurança para garantir que a pessoa com a BitCoin realmente seja o dono e ela exista.

Uma transação é requerida/ativada -> Um bloco que representa a transação é criado -> O Bloco é enviado para todos os "nós" a network -> Os "nós" validam a transação -> Os "nós" recebem recompensas pelo trabalho -> Um bloco é adicionado para a blockchain existente -> Transação completa

Simulação de Mineração

Ela é feita usando um valor baseado na dificuldade, então caso coloque dificuldade igual o resultado ainda será igual, pois o "hash", que é uma parte da blockchain, não muda na própria dificuldade
